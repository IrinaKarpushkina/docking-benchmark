# Docking Methods

## Supported Methods

### QVina
- **Description**: Fast version of AutoDock Vina
- **Input**: PDBQT files (protein and ligand)
- **Output**: PDBQT with docked poses
- **Metrics**: Affinity score

### AutoDock Vina
- **Description**: Classic docking method
- **Input**: PDBQT files (protein and ligand)
- **Output**: PDBQT with docked poses
- **Metrics**: Affinity score

### Boltz-2
- **Description**: Neural network-based docking with affinity prediction
- **Input**: YAML files with protein sequence and ligand SMILES
- **Output**: CIF structures, JSON with affinity
- **Metrics**: Affinity, confidence score

### DynamicBind (Stub)
- **Description**: Flexible docking with dynamic protein
- **Status**: Not yet implemented

### Uni-Mol (Stub)
- **Description**: Universal molecular model
- **Status**: Not yet implemented

### Interformer (Stub)
- **Description**: Transformer for interactions
- **Status**: Not yet implemented

### Gnina 1.3 (Stub)
- **Description**: Vina with CNN scoring
- **Status**: Not yet implemented

### PLAPT (Stub)
- **Description**: Protein-Ligand Affinity Prediction Transformer
- **Status**: Not yet implemented
